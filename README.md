# passgen
generator passwords

Using:

  in command line enter
  ```r
  $ passgen -p 10 -lus
  ```
  -p means password
  
  10 means numbers of symbols in password
  
  -lus or -lu 
  
  l = lower case, u = upper case, s = special symbols
  
  u cannot choose use numbers or not
  
  also u can use it just enter 
  ```r
  $ passgen
  ``` 
  in command line
